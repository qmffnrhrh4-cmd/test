# 모범답안 폴더 사용 가이드

이 폴더에 문제별 모범답안 파일을 저장하세요.

## 파일 형식

각 모범답안 파일은 다음 형식을 따라야 합니다:

```
[모범답안]
디지털변환 추진 전략 보고서

1. 추진배경
□ 4차 산업혁명기 디지털변환 가속화...
○ 구체적인 내용...

2. 현황 및 문제점
□ 기존 업무관행 문제...

3. 대응방안
□ 전사적 디지털 전환...

4. 향후계획
□ 단기/중장기 계획...

[필수 키워드]
디지털변환, 4차 산업혁명기, 지속가능성장, A기업, 경영기획본부, 빅데이터, AI(인공지능), 업무생산성·효율성, 수익성 개선, 사장 지시, 기존 업무관행, 경제 둔화, 로드맵, 추진전략, 현황 및 문제점, 대응방안, 향후계획

[금지어]
생각합니다, 것 같습니다, 아마도, 추측컨대, 하면 좋겠습니다

[채점 팁]
- 보고서 형식(제목, 1/2/3/4 대제목, □/○ 기호) 필수
- 제시자료의 단어를 그대로 사용했는지 확인
- 구체적인 수치나 날짜 포함 여부 확인
- 최소 15줄 이상 작성
```

## 파일 이름 규칙

- 한글 또는 영문으로 작성
- `.txt` 또는 `.md` 확장자 사용
- 예시: `디지털변환.txt`, `에너지위기.txt`, `신사업전략.md`

## 작동 방식

1. 사용자가 문제지를 업로드
2. 시스템이 이 폴더의 모든 파일을 스캔
3. 문제지 내용과 가장 잘 매칭되는 모범답안 파일을 자동 선택
4. 해당 파일의 모범답안, 키워드, 금지어를 불러옴
5. AI가 이 기준으로 학생 답안을 채점

## 매칭 알고리즘

시스템은 다음을 기준으로 매칭합니다:
- 모범답안의 첫 줄(제목)이 문제지에 포함되어 있는가?
- 필수 키워드가 문제지에 얼마나 포함되어 있는가?
- 모범답안 본문 일부가 문제지에 있는가?

## 주의사항

- 반드시 `[모범답안]`, `[필수 키워드]` 섹션은 포함해야 합니다
- 키워드는 쉼표(,)로 구분
- 금지어와 채점 팁은 선택사항
- 파일은 UTF-8 인코딩으로 저장

## 예시 파일

`예시_디지털변환.txt` 파일을 참고하세요.
