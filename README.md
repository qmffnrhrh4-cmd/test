## 📋 주요 기능

### 1️⃣ AI 답안 채점 ⭐⭐⭐ (완전히 새로워짐!)
- **문제별 맞춤 설정**:
  - 모범답안 직접 입력
  - 필수 키워드 직접 입력 (쉼표로 구분)
  - 금지어 직접 입력
  - 문제마다 다른 기준으로 채점 가능!

- **OPR 채점 전문가 AI**:
  - 20년 경력 수석 채점위원 역할
  - 실제 OPR 채점 기준 완벽 학습
  - 키워드 중심 채점 (실전과 동일)
  - 제시자료 단어 그대로 사용 여부 평가

- **초상세 피드백**:
  - ✅ 잘한 점 (구체적으로 어떤 부분)
  - ⚠️ 부족한 점 (어떤 키워드 누락)
  - 💡 개선 방법 (어떻게 고칠지)
  - 🎯 다음 학습 방향 (무엇을 공부할지)
  - 📊 키워드 매칭률 (12개 중 9개 매칭)

- **파일 첨부**: PDF, HWP, TXT 파일 첨부 가능

### 2️⃣ 실전 문제 생성 ⭐
- **폴더 기반 생성**: 특정 폴더의 자료들을 AI가 분석
- **실전 형식**: 실제 OPR과 동일한 문제 생성
- **10개 이상 제시자료**: CEO 메시지, 이메일, 메신저 등
- **파일 저장**: 생성된 문제를 TXT/JSON으로 저장

### 3️⃣ 공부 노하우
- 핵심 전략 TOP 5
- 채점 방식 이해
- 금지사항

### 4️⃣ API 설정
- Gemini API 키 입력
- 패키지 설치 안내

## 💡 사용 팁

### 답안 채점 (개선됨!)
1. **샘플 불러오기** 버튼 클릭 (처음 사용 시)
2. 학생 답안 입력 (또는 PDF/HWP 파일 첨부)
3. 모범답안 입력
4. 필수 키워드 입력 (쉼표로 구분: "키워드1, 키워드2, 키워드3")
5. 금지어 입력 (선택사항)
6. "✅ AI 채점 시작" 클릭
7. 10-30초 대기
8. 초상세 피드백 확인!

### 문제 생성
1. "📂 폴더 선택" → 문제지 또는 모범답안 폴더 선택
2. 난이도 선택
3. "✨ AI로 실전 문제 생성하기" 클릭
4. 생성된 문제 확인 및 저장

### ⚡ 꿀팁
- **매번 키워드 입력이 귀찮다면?** 샘플 불러오기로 예시를 보고, 복사해서 수정하세요!
- **API 키 없이도 사용 가능!** 기본 채점으로도 키워드 매칭이 작동합니다.
- **모범답안이 정확해야 AI 채점도 정확합니다!**

## ⚠️ FAQ

**Q: API 키 없이도 사용 가능한가요?**
A: 네! OPR_시스템.py를 사용하면 기본 채점 기능을 사용할 수 있습니다.

**Q: HWP 파일을 첨부하면 어떻게 되나요?**
A: TXT로 변환 후 사용을 안내합니다. (한글 → 다른 이름으로 저장 → TXT)

**Q: pip이 작동하지 않아요**
A: `python -m pip install` 명령을 사용하거나, 설치.bat을 관리자 권한으로 실행하세요.
