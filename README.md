# OPR 자동 채점 시스템 - AI 버전 🤖

Gemini AI 기반 OPR(논술형 시험) 자동 채점 시스템

**✨ Gemini AI로 상세한 피드백!**
**✨ PDF/HWP 파일 첨부 가능!**
**✨ 실전 문제 생성!**

더블클릭: **설치.bat**

### 2. 실행
더블클릭: **OPR_실행.bat**

**그게 다입니다! 👍**

## 📋 주요 기능

### 1️⃣ AI 답안 채점 ⭐
- **파일 첨부**: PDF, HWP, TXT 파일 첨부 가능
- **상세 피드백**:
  - ✅ 잘한 점 (구체적으로)
  - ⚠️ 부족한 점 (무엇이 문제인지)
  - 💡 개선 방법 (어떻게 고칠지)
  - 🎯 다음 학습 방향
- **정확한 채점**: Gemini AI가 모범답안과 비교
- **키워드 분석**: 매칭/누락 키워드 확인

### 2️⃣ 실전 문제 생성 ⭐
- **폴더 기반 생성**: 특정 폴더의 자료들을 AI가 분석
- **실전 형식**: 실제 OPR과 동일한 문제 생성
- **10개 이상 제시자료**: CEO 메시지, 이메일, 메신저 등
- **파일 저장**: 생성된 문제를 TXT/JSON으로 저장

### 3️⃣ 공부 노하우
- 핵심 전략 TOP 5
- 채점 방식 이해
- 금지사항

### 4️⃣ API 설정
- Gemini API 키 입력
- 패키지 설치 안내

## 🎯 AI 피드백 예시

```
【논리·정확성: 32/40점】

✅ 잘한 점:
   • 발전제약 해소, 법령 제개정 등 핵심 키워드를 포함함
   • 단기/중장기 구분이 명확함

⚠️ 부족한 점:
   • 전력망혁신위원회, WAMS 등 4개 키워드 누락
   • 구체적인 수치(2.6GW 등) 부족

💡 개선 방법:
   • 제시자료의 모든 수치를 포함하세요
   • 기술명(NWAs, WAMS 등)은 반드시 작성

🎯 다음 학습 방향:
   키워드 추출 연습에 집중하세요
```

## 🔧 설치 방법

### Windows

**방법 1: 자동 설치 (추천)**
```bash
설치.bat 더블클릭
```

**방법 2: 수동 설치**
```bash
python -m pip install google-generativeai PyPDF2 python-docx
```

### Gemini API 키 받기

1. https://makersuite.google.com/app/apikey 접속
2. API 키 생성
3. 프로그램에서 '⚙️ API 설정' 메뉴에서 입력

## 💡 사용 팁

### 답안 채점
1. PDF/HWP 파일 첨부 또는 직접 입력
2. "✅ AI 채점하기" 클릭
3. 상세한 피드백 확인

### 문제 생성
1. "📂 폴더 선택" → 문제지 또는 모범답안 폴더 선택
2. 난이도 선택
3. "✨ AI로 실전 문제 생성하기" 클릭
4. 생성된 문제 확인 및 저장

## ⚠️ FAQ

**Q: API 키 없이도 사용 가능한가요?**
A: 네! OPR_시스템.py를 사용하면 기본 채점 기능을 사용할 수 있습니다.

**Q: HWP 파일을 첨부하면 어떻게 되나요?**
A: TXT로 변환 후 사용을 안내합니다. (한글 → 다른 이름으로 저장 → TXT)

**Q: pip이 작동하지 않아요**
A: `python -m pip install` 명령을 사용하거나, 설치.bat을 관리자 권한으로 실행하세요.
