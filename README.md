## 📋 주요 기능

### 1️⃣ AI 답안 채점 ⭐⭐⭐ (완전히 새로워짐!)
- **🆕 완전 자동 채점**:
  - PDF/HWP 파일만 첨부하면 끝!
  - 문제 자동 인식 (AI 기반)
  - 모범답안, 키워드 자동 로드
  - 사용자는 파일 첨부만 하면 됩니다!

- **🆕 문제 데이터베이스**:
  - 문제, 모범답안, 채점기준 통합 관리
  - 답안 내용으로 문제 자동 매칭
  - 키워드 기반 스마트 인식 (신뢰도 표시)
  - 문제 DB 확장 가능 (문제_DB.json)

- **OPR 채점 전문가 AI**:
  - 20년 경력 수석 채점위원 역할
  - 실제 OPR 채점 기준 완벽 학습
  - Few-shot Learning으로 정확도 향상
  - 키워드 중심 채점 (실전과 동일)
  - 제시자료 단어 그대로 사용 여부 평가

- **초상세 피드백**:
  - ✅ 잘한 점 (구체적으로 어떤 부분)
  - ⚠️ 부족한 점 (어떤 키워드 누락)
  - 💡 개선 방법 (어떻게 고칠지)
  - 🎯 다음 학습 방향 (무엇을 공부할지)
  - 📊 키워드 매칭률 (12개 중 9개 매칭)

- **🆕 PDF 결과 생성**:
  - 채점 결과를 전문적인 PDF로 저장
  - 색상별 섹션 구분 (잘한 점/부족한 점/개선 방법)
  - 자동 페이지 나누기
  - 한글 폰트 자동 지원
  - 저장 후 자동으로 PDF 열기

### 2️⃣ 실전 문제 생성 ⭐
- **폴더 기반 생성**: 특정 폴더의 자료들을 AI가 분석
- **실전 형식**: 실제 OPR과 동일한 문제 생성
- **10개 이상 제시자료**: CEO 메시지, 이메일, 메신저 등
- **파일 저장**: 생성된 문제를 TXT/JSON으로 저장

### 3️⃣ 공부 노하우
- 핵심 전략 TOP 5
- 채점 방식 이해
- 금지사항

### 4️⃣ API 설정
- Gemini API 키 입력
- 패키지 설치 안내

## 💡 사용 팁

### 답안 채점 (🆕 완전 자동!)
**방법 1: 자동 채점 (추천!)**
1. "📂 PDF/HWP/TXT 파일 선택" 클릭
2. 학생 답안 파일 선택
3. 문제 자동 인식 알림 확인 (모범답안, 키워드 자동 입력!)
4. "✅ AI 채점 시작" 클릭
5. 10-30초 대기
6. 초상세 피드백 확인
7. "📄 PDF 저장" 버튼으로 전문 PDF 생성!

**방법 2: 수동 입력**
1. **샘플 불러오기** 버튼 클릭 (처음 사용 시)
2. 학생 답안 입력 (또는 PDF/HWP 파일 첨부)
3. 모범답안 입력
4. 필수 키워드 입력 (쉼표로 구분: "키워드1, 키워드2, 키워드3")
5. 금지어 입력 (선택사항)
6. "✅ AI 채점 시작" 클릭
7. 10-30초 대기
8. 초상세 피드백 확인!

### 문제 생성
1. "📂 폴더 선택" → 문제지 또는 모범답안 폴더 선택
2. 난이도 선택
3. "✨ AI로 실전 문제 생성하기" 클릭
4. 생성된 문제 확인 및 저장

### ⚡ 꿀팁
- **🆕 파일만 첨부하면 끝!** 문제가 자동으로 인식되어 모범답안과 키워드가 자동 입력됩니다!
- **🆕 결과를 PDF로 저장!** 전문적인 PDF 형식으로 채점 결과를 저장하고 공유하세요!
- **문제 DB 확장 가능!** 문제_DB.json에 새로운 문제를 추가하면 자동 인식됩니다!
- **API 키 없이도 사용 가능!** 기본 채점으로도 키워드 매칭이 작동합니다.
- **모범답안이 정확해야 AI 채점도 정확합니다!**

## ⚠️ FAQ

**Q: API 키 없이도 사용 가능한가요?**
A: 네! OPR_시스템.py를 사용하면 기본 채점 기능을 사용할 수 있습니다.

**Q: HWP 파일을 첨부하면 어떻게 되나요?**
A: TXT로 변환 후 사용을 안내합니다. (한글 → 다른 이름으로 저장 → TXT)

**Q: pip이 작동하지 않아요**
A: `python -m pip install` 명령을 사용하거나, 설치.bat을 관리자 권한으로 실행하세요.
