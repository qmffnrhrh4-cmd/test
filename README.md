# OPR 자동 채점 시스템 📚

OPR(논술형 시험) 자동 채점, 문제 생성, 학습 가이드를 제공하는 **올인원** 시스템입니다.

**✨ 모든 기능이 하나의 파일로 통합되었습니다!**
**✨ 추가 설치 불필요! Python만 있으면 바로 실행!**

## 🚀 빠른 시작

### Windows
더블클릭: **OPR_실행.bat**

### Mac/Linux
```bash
python3 OPR_시스템.py
```

**그게 다입니다! 👍**

---

## 📋 주요 기능

### 1️⃣ 답안 자동 채점
- 키워드 기반 자동 채점
- 금지어 감점 처리 (-2점/개)
- 논리·정확성 (40점) + 명확·간결성 (30점) + 완결성 (30점)
- 상세 피드백 제공

### 2️⃣ 연습 문제 생성
- 난이도별 문제 (쉬움/보통/어려움)
- 실제 OPR 형식
- 키워드 자동 제공
- 파일 저장 기능

### 3️⃣ 공부 노하우
- 핵심 전략 TOP 5
- 채점자 관점 팁
- 금지사항 안내

### 4️⃣ 학습 계획
- 4주 단계별 계획
- 주차별 목표 및 활동
- 체크포인트

### 5️⃣ 시험 당일 체크리스트
- 12가지 확인사항
- 클릭 체크 기능
